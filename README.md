## Usage

JPEG 編碼：

* 將 image 放到 `./pics` 的資料夾
* 運行 `JPEGEncoding.py`
* 壓縮後的結果會放在 `./jpeg` 的資料夾

JPEG 解碼：

* 確認壓縮後的結果放在 `./jpeg` 的資料夾
* 運行 `JPEGDecoding.py`
* 會 imshow 出解碼後的 image
